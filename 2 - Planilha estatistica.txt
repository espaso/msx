Lógica dos cálculos (resumo técnico)
True Range (TR) = max(High−Low, |High−FechPrev|, |Low−FechPrev|)
ATR(n) = média móvel do TR em n períodos (Parâmetros!B4).
Ret Log = LN(Fechamento / FechPrev)
MédiaN / DesvN (n) sobre Fechamento com janela Parâmetros!B5.
ZScore = (Fechamento − MédiaN) / DesvN
Slope(n) = SLOPE(Fechamento_últimos_n, Índice_últimos_n) com n = Parâmetros!B6.
VolMediaN / VolRel com janela Parâmetros!B7.
Bandas ±2σ para referência estatística (não são sinais por si sós).
Observação técnica: as fórmulas estão em inglês (padrão OOXML) e usam vírgula como separador de argumentos. Isso é o correto para arquivos salvos por automação — o Excel converte/localiza automaticamente na sua interface PT‑BR.

Dicas de leitura (educacional)
ATR: dimensione alvos e stops com múltiplos do ATR compatíveis com o regime de volatilidade do dia.
Z-Score: valores |Z| > 2 marcam zonas atípicas; use como alerta, não como gatilho único.
Slope > 0 ou < 0 reforça tendência no recorte da janela.
VolRel > 1 sugere fluxo acima da média — útil para filtrar rompimentos.
